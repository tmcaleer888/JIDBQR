The Next FaceBook and the Next Google Combined

This program takes two user input names (or just words), calculates their value by summing the Unicode value for each letter, then takes the average of the two sums. From this new value, the code searches through a library of names to find a name that has the same summed value. If there is no exact match, the closest name is provided, with the difference indicated. Brought to you by me and Google Gemini.

Use for all of your naming needs.

---------------------

To run in terminal:

python main.py
or
python3 main.py
