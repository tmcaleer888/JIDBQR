The Next FaceBook and the Next Google Combined

This program received two user input names (or just words), calculates their value by adding pu the Unicode value for each letter, then takes the average of the two inputs. From this new value, the code searches through a library of names to find a names that has the same total letter value. If there is no exact match, the closest name is provided, with the difference indicated. Brought to you by me and Google Gemini.

To run in terminal:

python main.py
or
python3 main.py
